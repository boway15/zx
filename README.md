# 朴素自习室 v0.9.1

**斯是陋室** — 面向务实学习、工作者的自助自习空间。

美团/抖音购卡 → 管理员核销发兑换码 → 用户**免登录**输入兑换码 → 预约座位 → 开门。

访问 http://localhost:9080

## 用户流程（无需登录）

1. 打开 H5，输入 **11 位数字兑换码**
2. 确认卡类型、兑换期限或使用有效期
3. 「预约座位」Tab：选择 301–305 教室座位（首次预约后激活会员卡）
4. 「使用」Tab：一键开门 / 查看门锁密码
5. 同一兑换码在有效期内可重复输入进入

## 教室与座位

| 教室 | 座位数 |
|------|--------|
| 301 | 7 |
| 302 | 11 |
| 303 | 7 |
| 304 | 6 |
| 305 | 7 |

预约规则：日卡按天预约；周卡/月卡可一次固定有效期内剩余天数的同一座位。

## 管理后台 `/admin/login`

- **核销兑换**：美团/抖音券码核销后生成 11 位兑换码
- **座位管理**：设置各座位可约 / 不可约
- **预约查看**：按日期或兑换码查询；取消预约仅影响单日
- 卡价、开门记录、系统设置、通通锁配置

默认管理员：`admin` / 见 `.env` 中 `ADMIN_DEFAULT_PASSWORD`

## 生产环境注意事项

上线前请确认：

- `NODE_ENV=production`
- `DB_SYNC=false`
- `WECHAT_MOCK_PAY=false`
- `TTLOCK_MOCK_UNLOCK=false` 且已配置真实 `TTLOCK_LOCK_ID`
- 使用强随机 `JWT_SECRET` / `ADMIN_JWT_SECRET`

## 启动

```bash
docker compose up -d --build
```
